# NFT Card Component Challenge

The challenge involves building an NFT card component with a focus on clean UI, accessibility, and hybrid rendering performance using Next.js with AMP support.

This folder contains the implementation for the **seventeenth challenge** from [Frontend Mentor](https://www.frontendmentor.io).  
The goal is to create a visually appealing NFT preview card, optimized for performance and mobile-first design.

## Technologies & Tools

- **Next.js (Page-Routed ISR):**  
  Enables fast, scalable rendering with Incremental Static Regeneration and hybrid AMP support.

- **AMP (Accelerated Mobile Pages):**  
  Ensures lightning-fast performance and mobile optimization for hybrid app delivery.

- **TypeScript:**  
  Provides type safety and improved developer experience.

- **Tailwind CSS:**  
  Utility-first CSS framework for rapid styling and responsive layout.

- **BEM Methodology:**  
  Block Element Modifier naming convention for maintainable and scalable CSS structure.

- **Vercel:**  
  Used for deploying the project with seamless integration and global CDN.
